# 写作页迭代 · Jobs-to-be-Done

> 框架：Christensen JTBD + Value Proposition Canvas（jobs-to-be-done skill）
> 输入锁定（2026-06-07）：persona = 随笔为主 + 轻创作延伸；主问题 = ①捕捉红线 + ②写作-分析闭环
> 状态：待校验草稿（基于问题陈述 + 竞品研究；`emotional jobs` 需你点头是否真实）

---

## Context / Situation（job 在什么情境下发生）

- **随笔情境**：当我有情绪 / 想法涌上来，或一天结束想梳理时 → 我打开应用想把它记下来
- **轻创作情境**：当某段随笔我觉得「这个值得认真写」时 → 我想把它打磨成一篇成稿
- **今天用什么替代**：备忘录 / flomo / Notion / 纸本 / 微信文件传输助手 —— 这些才是真正的竞争对手，不是网文码字软件

---

## 1. Customer Jobs

### Functional Jobs（功能性 · 要完成的任务）
| Job | 随笔 | 轻创作 |
|---|:--:|:--:|
| 低摩擦地把脑子里的想法/情绪**快速落成文字** | ●核心 | ○ |
| **随时停、随时接着写**，中途不丢 | ●核心 | ● |
| 回头能**找到、并看见自己的变化** | ●核心 | ○ |
| 把一段随笔**扩展/重写成一篇成稿** | — | ●核心 |
| 给成稿做**轻量结构**（开头/展开/收束，非网文大纲） | — | ●核心 |
| 把成果**导出/分享** | ○ | ●核心 |

### Social Jobs（社会性 · 想被怎么看）
- 轻创作：分享成果时显得「有质感、会写」→ 体面的导出物（呼应竞品「导出长图」+ 你的 ljg-card）
- 随笔：几乎不对外，但有一种**对未来的自己负责**的准社交——「写给以后会回看的我」

### Emotional Jobs（情感性 · 想要 / 想避免）
**想要：**
- 进入状态、**被陪伴**的感觉（不是对着空框单打独斗）
- 落笔后**被看见、有意义**的回声
- 看到自己**坚持下来**的成就感
- **安全感**：内容不会丢、私密不会被偷/被喂 AI

**想避免：**
- 空白页焦虑（不知从何写起）
- 写完**石沉大海的虚无**
- 怕私密文字泄露 / 被拿去训练
- 断更后的**愧疚感**

## 2. Pains（按强度排序 ★）

| 痛点 | 类型 | 强度 |
|---|---|:--:|
| **写完无即时回馈**——落笔当下感受不到画像/情感/热力分析，智能全堆在写完之后 | Unresolved（没人解决） | ★★★ 最致命 |
| **内容易丢 / 无自动保存**——刷新、误触就没了 | Challenge | ★★★ 信任红线 |
| **空白框无从下手**——没有引导/选题 | Challenge | ★★ 高频 |
| 要看分析得**先写完再跳到别的页**，路径割裂 | Costliness | ★★ |
| 半成品散落、找不回、接不上 | Common mistake | ★★ |
| 写时易分心、进不去状态 | Challenge | ★ |

> 「写完无即时回馈」是**唯一别人没在解决**的痛——所有竞品写作页都是哑输入框。这正是问题②，也是你区别于「备忘录/flomo」的根本理由。

---

## 3. Gains（按必要性排序）

| 收益 | 类型 | 等级 |
|---|---|:--:|
| 保存即出「**今日心绪卡 + 和过去的呼应**」——落笔立刻有意义回声 | Expectation | must-have（闭环亮点） |
| **无感自动保存** + 草稿永不丢 | Saving | must-have（红线） |
| 打开写作页**就有一句引导/选题**，化解空白页 | Expectation | must-have |
| 省去「写完再翻分析」的跳转，**当页闭环** | Saving | high |
| 「自带 Key、你的文字只走你自己的 Key」**隐私可见化** | Adoption | high（差异化定位） |
| 从随笔**一键长成一篇**的顺滑路径 | Adoption | medium（轻创作，可后置） |
| 专注/沉浸写作模式 | Life improvement | medium |

---

## 4. 收口：随笔 job vs 轻创作 job（避免被网文带偏）

- **这轮聚焦随笔核心 job**：快速捕捉 / 不丢 / 落笔即回馈 / 看见变化 —— 直接对应已锁定的问题①②
- **轻创作 job 留路径、不铺重功能**：保留「随笔 → 顺手长成一篇 → 体面导出」的延伸口，但**不做**网文式大纲/角色/伏笔/敏感词工作区
- **明确不做**（来自竞品但不契合）：强制小黑屋、重排版引擎、关系图谱/故事地图、取名器
