# 文字时光机 · 迭代二设计文档

**日期**：2026-05-01  
**状态**：已确认，待实现

---

## 背景与目标

文字时光机目标从个人工具扩展为面向公众的产品。核心用户是**写作爱好者**和**自我探索者**，共同动机是：写作是手段，自我理解是目的。

本次迭代聚焦两件事：
1. 建立有文学质感的视觉设计语言
2. 解决冷启动阶段的空状态体验问题

分发策略：先打磨产品本身（MVP），后续考虑打包为桌面 App。

---

## 一、视觉语言系统

### 色板
| 用途 | 色值 |
|------|------|
| 页面底色 | `#F7F4EF`（温暖米白） |
| 主文字 | `#2C2C2C`（近黑） |
| 强调色 | `#8B6F47`（暖棕，墨水感） |
| 辅助/边框 | `#C9B99A`（浅棕） |
| 卡片背景 | `#FFFFFF` |

### 字体
- **标题**：Noto Serif SC（衬线，文学质感），fallback Georgia
- **正文**：PingFang SC / 系统无衬线（保证可读性）

### 质感细节
- 卡片边框：`1px solid #E8E0D5`，不堆叠阴影
- 按钮圆角：4px（精致而非圆润）
- 分隔线：极浅色细实线，模拟纸张感

## 二、冷启动方案

功能**不做锁定**，始终展示全部功能入口。内容不足时给温和提示，激励而非阻断。

### 0 篇状态
总览页显示欢迎引导卡，替代空白图表：
> "你好，这里是你的文字时光机。写下第一篇随笔，开始记录属于你的文字轨迹。"

配大号"写第一篇"按钮直达写作页。

### 1-2 篇状态
显示现有分析内容，词云/情感曲线等功能下方嵌入提示条：
> "写得越多，分析越准——每一篇都让画像更清晰。"

### 3 篇以上
总览页顶部出现**"今日回顾"卡片**：随机浮现一篇旧文开头几句，配写作日期和情感分，制造重读旧文的惊喜感。

## 三、核心页面改版范围

### ① 总览页（优先级最高）
- 今日回顾卡片（3篇+时显示）
- 0篇欢迎引导卡
- 统计卡片重设计：数字更大、加暖棕色点缀
- 随笔列表：左侧情感色条（积极=暖棕、消极=灰蓝、平静=米色）
- 内容不足提示条

### ② 写作页（优先级中）
- 编辑区扩展全宽，去除多余边框感
- 字数统计移至右下角
- 标题/日期输入框样式统一

### ③ 写作画像页（优先级中）
- 配色跟随新设计语言（暖棕色系）
- 深度解读结果区加纸张感背景

### 详情页
跟随新设计语言做同步样式调整，不单独重做。
